História dos computadores

Primeira Geração

A primeira geração foi marcada pelos computadores que utilizavam válvulas e pelo tamanho dos computadores, podendo ocupar desde uma sala até o andar de um prédio.

Sua função principal era a realização de cálculos para fins militares.

O que são Clients?

Clients são responsáveis por solicitar informações (serviços) dos servidores, podemos citar como exemplo os dispositivos moveis, aplicações web, navegadores, etc.

Em computação, um cliente é uma peça de hardware ou software de computador que acessa um serviço disponibilizado por um servidor como parte do modelo cliente-servidor de redes de computadores.

Um cliente geralmente não compartilha nenhum de seus recursos, mas solicita conteúdo ou serviço de um servidor. Os clientes, portanto, iniciam sessões de comunicação com os servidores, que aguardam solicitações recebidas.

O que são Servers?

Server são responsáveis por fornecer informações (serviços) para os clientes, podemos citar como exemplo os servidores de arquivos, firewall, web, etc.

O servidor geralmente (mas nem sempre) está em outro sistema de computador, caso em que o cliente acessa o serviço por meio de uma rede.

Um host de servidor executa um ou mais programas de servidor, que compartilham seus recursos com clientes.

Como é uma estrutura básica de um HTML?

O HTML é uma linguagem de marcação de hipertexto, sua função é definir o significado e a estrutura do conteúdo da página. Ele tem uma função semântica, informando aos navegadores o que cada bloco de texto quer representar e como exibir o conteúdo na página.

O HTML é composto por Tags, para formar uma Tag usamos o sinal de menor "<" e maior ">", dentro dela temos o nome e as propriedades que podemos ou não colocar em nossos elementos.

A estrutura básica do html pode ser observada abaixo:

               
                  <!DOCTYPE>
                  <html lang="pt-br">
                     <head>
                     </head>
                     <body>
                        <header>
                        </header>

                        <main>
                        </main>

                        <footer>
                        </footer>
                     </body>
                  </html>

Entendendo a diferença entre listas ordenadas e não ordenadas

Listas ordenadas possuem uma estrutura formada com uma numeração antes dos elementos.

O código abaixo é uma lista ordenada de 3 itens:

               
                  <ol>
                     <li>Item 1</li>
                     <li>Item 2</li>
                     <li>Item 3</li>
                  </ol>

Resultado visto no navegador:

  1. Item 1
  2. Item 2
  3. Item 3

Listas não ordenadas possuem uma estrutura formada com sem uma numeração antes dos elementos.

O código abaixo é uma lista ordenada de 3 itens:

               
                  <ul>
                     <li>Item 1</li>
                     <li>Item 2</li>
                     <li>Item 3</li>
                  </ul>

Resultado visto no navegador:

  • Item 1
  • Item 2
  • Item 3
